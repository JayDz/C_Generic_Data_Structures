Function: Performs a linear search on base. If key is found, the return value equals its index. Otherwise, the return value is -1.

int jd_lsearch(void* base, void* key, size_t elem_size, size_t length, int(*cmp)(void*,void*));

Parameters:

void* base: A pointer to an array.
void* key: A pointer to the key we're to search for.
size_t elem_size: The size of the elements stored in array.
size_t length The length of array.
int(*cmp)(void*,void*): A pointer to a function receiving two elements to compare. Return zero to indicate elements are equal. If NULL, memory blocks are compared.

Return Value:

The index of key or -1 if not found.

Example Usage:

 
#include "JDSearch.h"

/* Create an array and search for a key. */
#define SIZE 7

int cmp(void* a, void* b)
{
	return *((int*)a) - *((int*)b);
}

int main(void)
{
	int arr[SIZE] = {10,20,30,40,50,60,70};
	int key = 50;
	int key_index = -1;
	key_index = jd_lsearch(arr, &key, sizeof(int), SIZE, &cmp);
    
	return 0;
}
