JD Developer Documentation

Containers:  
JDArrayStack.h A Stack implemented with a dynamic array underneath.
JDStack.h A Stack implemented with a singly linked list underneath.
JDArrayQueue.h A Queue implemented with a dynamic array underneath.
JDQueue.h A Queue implemented with a singly linked list underneath.
JDDeque.h A Deque implemented with a doubly linked list underneath.
JDForwardList.h An implementation of a singly linked list.
JDList.h An implementation of a doubly linked list.
JDSet.h An implementation of Set.
   
Algorithms:  
JDSearch.h Implementation of traditional search algorithms such as Linear Search and Binary Search.
JDSort.h Implementation of traditional sorting algorithms such as Bubble sort, Insertion sort, Selection sort, Mergesort, and Quicksort.
   
Utility:  
JDError_codes.h Defines contants used by various containers and algorithms to indicate whether an operation succeeded or failed. Also defines true and false constants.
