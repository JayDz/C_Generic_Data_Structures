JDError_codes

JD_FMALLOC: Returned when a JD function failed to allocate heap memory.
JD_NULLPTR: Returned when a pointer to a JD data structure is NULL.
JD_NULLPTR_ARG: Returned when a pointer argument sent to a JD function is NULL. E.g push, pop, enqueue, dequeue all require an extra argument pointing to a value or a destination to store a value.
JD_EMPTY: Returned when client code attempts to query an empty data structure.
JD_SUCCESS: Returned when operation succeeded.
JD_NOTFOUND: Returned when an item was not found in collection.
